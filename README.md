## Descrição

Esse projeto consiste na construção de uma tabela que contempla capitais brasileiras. Nessa tabela está a distância entre essas capitais. 

* Aracajú
* Belém
* Belo Horizonte
* Brasília
* Campo Grande
* Cuiabá
* Curitiba
